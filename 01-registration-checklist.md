## Step-by-step registration

- [ ] **Create / log in** to Hack2skill account: https://hack2skill.com/login
- [ ] **Open event page:** https://hack2skill.com/event/bah2026/
- [ ] Click **Register Now** and complete team profile (3–4 members, all enrolled at Indian institutions)
- [ ] **Select Problem Statement 8:**
  > Detection and Characterization of Subsurface Ice in Lunar South Polar Regions Using Chandrayaan-2 Radar and Imagery Data for Landing Site and Rover Traverse Planning
- [ ] Verify each team member's eligibility (UG / PG / PhD; no working professionals)
- [ ] **Join Discord** via event page — channel announcements and mentor access
- [ ] Email support if needed: support+isrobah2026@hack2skill.com
